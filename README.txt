Mohit & Nidhi 6-Month Anniversary Site
--------------------------------------

What you need to do:

1) MUSIC
   - Put your main song here as:   assets/music/main-song.mp3
   - Put your gift song here as:   assets/music/gift-song.mp3

2) PHOTOS / VIDEOS
   - Put June photos/videos in:       assets/photos/june/
   - July:                            assets/photos/july/
   - August:                          assets/photos/august/
   - September:                       assets/photos/september/
   - October:                         assets/photos/october/
   - November:                        assets/photos/november/

   Match the file names used inside index.html (e.g. june-photo1.jpg, june-video1.mp4)
   OR edit the paths in index.html to match your own file names.

3) GIFs
   - There are placeholder GIF files inside assets/gifs/:
       bear-default.gif
       june-bear.gif
       july-bunny.gif
       august-stars.gif
       september-bear.gif
       october-moon.gif
       november-stars.gif
       gift-bunny.gif

   Replace each with your own cute bear/bunny/star GIFs, keeping the same file names.

4) LETTER
   - Open index.html in a code editor.
   - Search for: "const letterContent =" and "const letterPS =".
   - Replace that text with your own message for Nidhi.
